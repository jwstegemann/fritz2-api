This component generates a single radio. So this component supports the use case to select or deselect an option. In general prefer a checkbox for this use case. If an application has a strong focus on mobile, consider a switch instead!

This component mostly exist only as base item for a radioGroup!

You can set different kind of properties like the label or different styling aspects like the colors of the background, the label or the checked state. Further more there are configuration functions for accessing the checked state of this box or totally disable it.

Example usage

val cheeseStore = storeOf(false)
radio(value = cheeseStore) {
     label("with extra cheese") // set the label
}
